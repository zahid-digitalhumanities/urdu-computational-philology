# Urdu Computational Philology Toolkit

A Python-based toolkit for the computational analysis and digital preservation of classical Urdu poetry. This repository implements the reproducible methodology presented in the following HEC-recognized (Y-category) research:

"Computational Framework for the Descriptive Analysis and Digital Preservation of Classical Urdu Poetry"  
Muhammad Zahid, Khalid Saeed Siddiqui, Naima Mubeen, Arslan Ejaz.  
Liberal Journal of Language & Literature Review, 4(1), 2026.  
ISSN: 3006-5887  
Publication link: https://llrjournal.com/index.php/11/article/view/468

# Research Objectives

This toolkit addresses critical gaps in Digital Humanities research for non-Latin script literature:

- Right-to-left (RTL) processing of Urdu Nastaliq script  
- Low-resource language workflows requiring minimal computational infrastructure  
- Transparent, rule-based methodologies instead of black-box AI models  
- Digital preservation of classical Urdu poetic corpora  
- Reproducible descriptive analytics for literary and linguistic scholarship
